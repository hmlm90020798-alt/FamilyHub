# FamilyHub PWA

Este é o teu projeto de app familiar.

## Como colocar a funcionar:
1. Cria um projeto no [Firebase Console](https://console.firebase.google.com/).
2. Ativa o **Cloud Firestore**. Nas 'Rules', coloca em modo de teste para facilitar o início.
3. Regista uma aplicação Web e copia as credenciais (`firebaseConfig`).
4. Abre o ficheiro `index.html` e cola essas credenciais onde diz 'SUBSTITUA PELAS SUAS CONFIGURAÇÕES'.
5. Sobe todos os ficheiros para um repositório no GitHub.
6. Em 'Settings' > 'Pages', ativa a publicação.

Pronto! Agora podes abrir o link no telemóvel e 'Adicionar ao Ecrã Principal'.
